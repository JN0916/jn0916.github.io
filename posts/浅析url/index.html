浅析URL

什么是URL？

URL(Uniform Resource Locator)统一资源定位符是因特网上标准的资源的地址,如同在网络上的门牌。它最初是由蒂姆·伯纳斯-李发明用来作为万维网的地址。

URL= 协议 + 域名/IP + 端口 + 路径 + 查询参数 + 锚点

DNS 的作用是什么，nslookup 命令怎么用

DNS 的全称是 Domain Name System 或者 Domain Name Service，它主要的作用就是将人们所熟悉的网址 (域名) “翻译”成电脑可以理解的 IP 地址，这个过程叫做 DNS 域名解析。

通过nslookup命令可以查看域名对应的ip地址：

nslookup baidu.com

IP 的作用是什么，ping 命令怎么用

IP的主要作用就是2个：

  1. 如何定位一台设备。
  2. 如何封装数据报文，以跟其他设备交流。

IP分为内网与外网，中间通过路由器将其联通。每个路由器都有自己独立的外网IP，可以用过ip138.com查询。不同设备连接同一路由器，通过内网IP分发。

特殊的IP：

  • 127.0.0.1 表示自己
  • localhost 表示通过hosts指定为自己
  • 0.0.0.0 不表示任何设备。

ping的用法可以在命令行中输入ping + 域名/ip。可以得到域名的解析记录。

域名是什么，分别哪几类域名

域名就是对IP的别称。

  1. 多个域名可以对应同一个IP
  2. 一个域名也可以对应多个IP（负载均衡）

域名分为顶级域名和二级域名；顶级域名又可分为国际域名和国内域名；二级域名分为类别域名和行政区域名。
