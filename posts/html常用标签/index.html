HTML 常用标签

HTML(HyperText Markup Language)是一种是一种用于创建网页的超文本标记语言。

这里先简单介绍以下三中标签

a 标签

作用：a 元素可以创建一个到其他网页、文件、同一页面内的位置、电子邮件地址或任何其他 URL 的超链接

属性：

  1. href:可以跳转到某个指定的地址或者文件
<a href="http://qq.com">QQ</a>
  1. target:该属性指定在何处显示链接的资源。target 属性的四个关键字具有特殊的意义：
  • _self ：加载当前页面，如果没有指定属性，则此值为默认；
  • _top： 加载响应进入顶层页面，如果没有父级框架或页面，效果与 _self 相同；
  • _parent：在父级页面打开地址；
  • _blank：在新窗口打开；

img 标签

作用：发出 get 请求，展示一张图片

属性：

  1. alt：图像的替代文本；
  2. src：显示图像的 URL；
  3. height：图像的高度；
  4. width：图像的宽度；

img 响应式：适用于多种设备查看图片

img {
max-width: 100%;
    }

table 标签

作用：定义一个表格，以以下标签共同组成 table thead tbody tfoot tr td th

<table>
<thead>
<tr>
<th>表头1</th>
<th>表头2</th>
<th>表头3</th>
</tr>
</thead>
<tbody>
<tr>
<td>内容1</td>
<td>内容2</td>
<td>内容3</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>脚注1</td>
<td>脚注2</td>
<td>脚注2</td>
</tr>
</tfoot>
</table>
